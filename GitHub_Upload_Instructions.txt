# How to Upload Your Project to GitHub (Step-by-Step for Beginners)

Don't worry if you have 0 knowledge about GitHub! Follow these exact steps, one by one.

---

### STEP 1: Create a Repository (A Folder) on GitHub.com
1. Open your web browser (Chrome, Edge, etc.) and go to: https://github.com/
2. Log in to your account.
3. In the top-right corner of the screen, next to your profile picture, click the **+ (Plus)** icon and select **New repository**.
4. You will see a page to create a new repository.
   - In the **Repository name** box, type exactly: `OilGas_Forecaster`
   - Leave the "Description" box blank.
   - Make sure **Public** is selected (or Private if you don't want anyone else to see it).
   - **CRITICAL:** DO NOT check any boxes under "Initialize this repository with". Do NOT check "Add a README file". 
5. Scroll to the bottom and click the green **Create repository** button.
6. You will be taken to a new page. Look for a section that says **"…or push an existing repository from the command line"**.
7. Under that section, you will see a box with code that looks something like this:
   `git remote add origin https://github.com/RAGHAV0018/OilGas_Forecaster.git`
8. Keep this web page open. You will need that link in Step 3.

---

### STEP 2: Open the Correct Terminal on Your Computer
1. You are currently using an IDE (like VS Code or Cursor). Look at the bottom part of your screen. 
2. You should see a panel with tabs like "Problems", "Output", "Debug Console", and **"Terminal"**.
3. Click on the **Terminal** tab. 
4. In the terminal, make sure your prompt (the text on the left before you type) ends with `OilGas_Forecaster>`. 
   *(If it says `4th sem main EL>`, type `cd OilGas_Forecaster` and press Enter to go into the correct folder).*

---

### STEP 3: Run the Git Commands
Now, you will type these commands into that Terminal, one by one. Press the **Enter** key after each command.

1. Tell Git to prepare all the files we created:
   Type: `git add .`
   Press Enter. *(Note: there is a space and a dot after the word add)*

2. Save these files with a message:
   Type: `git commit -m "Initial commit: Added NLP Model and Requirements"`
   Press Enter.

3. Rename the main branch (standard practice):
   Type: `git branch -M main`
   Press Enter.

4. **Connect your computer to GitHub:**
   Go back to the GitHub webpage you left open in Step 1. Copy the line that starts with `git remote add origin...` and paste it into the terminal. It should look like this:
   Type: `git remote add origin https://github.com/RAGHAV0018/OilGas_Forecaster.git`
   Press Enter.

5. **Upload the code:**
   Type: `git push -u origin main`
   Press Enter.

*Note: If a pop-up appears asking you to sign in to GitHub, click "Sign in with your browser" and authorize it.*

---

### STEP 4: Check if it Worked!
1. Go back to your browser where GitHub is open.
2. Refresh the page.
3. You should now see your files (`requirements.txt`, `src/nlp_scorer.py`, etc.) on the screen! You have successfully backed up your code to GitHub.
